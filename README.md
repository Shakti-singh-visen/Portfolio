## 🚀 Getting Started

### 📋 Prerequisites

Before setting up the project locally, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v18.0.0 or higher recommended)
- [npm](https://www.npmjs.com/)

### 🔧 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Shakti-singh-visen/Portfolio.git
   cd Portfolio
   ```

2. **Install project dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```
   The local preview will be available at [http://localhost:5173/](http://localhost:5173/).

4. **Build for production**:
   ```bash
   npm run build
   ```
   This generates optimized static files in the `dist/` directory ready for deployment.
